# ProjectWolfGame
A prototype for a terminal-based RPG.

## Dependencies
* Boost Libraries - http://www.boost.org/

## How to Build
1. Add environment variables BOOST_ROOT and BOOST_LIBRARYDIR
2. Open visual studio --> build solution

## Controls

* ~12345 - Menu access
* wasdqe - Movement
* i - inventory
* Space - Attack
* t - Change target
* F5 - Save
* F9 - Load
